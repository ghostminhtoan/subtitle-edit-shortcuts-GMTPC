# Profile 02: Translate - Existing Subtitle
## (Dịch từ phụ đề đã có timing sẵn)

> **Workflow:** Mở file sub có sẵn → Bật Translation Mode → Dịch → Save
> **Dành cho:** File sub đã có timing chuẩn, chỉ cần nhập bản dịch.

## 💡 TIPS SỬ DỤNG

1. **Bước 1:** Mở file sub có sẵn → `Ctrl+Shift+O` để bật Translation Mode
2. **Flow dịch chuẩn:** `Alt+Down` (nghe + đọc bản gốc) → Dịch vào hộp dưới → `Alt+Down` tiếp
3. **Auto-translate:** `Ctrl+Shift+G` để dịch tự động, sau đó chỉnh sửa lại cho chuẩn
4. **Switch boxes:** `Ctrl+Alt+O` để chuyển focus giữa bản gốc và bản dịch
5. **Kiểm tra:** `Alt+F7` spell check, `Ctrl+Shift+F` fix common errors trước khi save
6. **Merge dòng:** `Ctrl+M` gộp với dòng trước, `Alt+K` gộp & không ngắt dòng
