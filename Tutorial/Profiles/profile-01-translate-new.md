# Profile 01: Translate - New Subtitle
## (Tạo phụ đề mới từ đầu)

> **Workflow:** Nghe video → Tạo timing → Nhập văn bản → Dịch
> **Dành cho:** Người làm sub từ đầu, chưa có file phụ đề nào.

---

## 💡 TIPS SỬ DỤNG

1. **Bắt đầu từ đầu:** Mở video → `Ctrl+N` tạo file mới → Dùng `Alt+N` để tạo sub đầu tiên
2. **Flow làm việc chuẩn:** Nghe → `Alt+N` → Nhập text → `Ctrl+R` ngắt dòng → `Ctrl+D0` sang dòng sau
3. **Dùng Waveform:** `Insert` để chèn sub từ waveform, dùng `Ctrl+Oemplus (=)` / `Ctrl+OemMinus (-)` để zoom
4. **Chế độ dịch:** `Ctrl+Shift+O` để bật translation mode, `Ctrl+Alt+O` để switch giữa 2 hộp
5. **Kiểm tra:** `Alt+F7` spell check, `Ctrl+Shift+F` fix common errors
6. **Save thường xuyên:** `Ctrl+S` để lưu, `Ctrl+Shift+S` để save as
