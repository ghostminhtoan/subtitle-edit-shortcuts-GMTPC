# 2. Hướng dẫn dịch phụ đề từ file subtitle có sẵn

Nếu bạn đã có file phụ đề bằng tiếng nước ngoài và muốn dịch nó sang tiếng Việt (hoặc ngôn ngữ khác), Subtitle Edit có công cụ giúp bạn làm điều này nhanh chóng. Hướng dẫn này dành cho người mới bắt đầu, từng bước một.

## Chuẩn bị
- Đảm bảo bạn đã cài đặt Subtitle Edit (từ file 1).
- Có file phụ đề gốc (thường có đuôi .srt hoặc .ass).

## Các bước dịch phụ đề

### Bước 1: Mở Subtitle Edit
- Nhấp đúp vào biểu tượng Subtitle Edit trên máy tính để mở phần mềm.

### Bước 2: Tải file phụ đề hiện tại
- Ở menu trên cùng, chọn **File** > **Open** (hoặc nhấn Ctrl+O).
- Tìm và chọn file phụ đề của bạn (ví dụ: movie.srt).
- Phụ đề sẽ hiện ra ở cửa sổ chính, với từng dòng có thời gian hiển thị.

### Bước 3: Sử dụng công cụ dịch
- Chọn **Tools** > **Translate** từ menu.
- Một cửa sổ mới sẽ mở ra với các tùy chọn dịch.

### Bước 4: Chọn ngôn ngữ
- Ở phần "Translate from" (dịch từ), chọn ngôn ngữ của phụ đề gốc (ví dụ: English).
- Ở phần "Translate to" (dịch sang), chọn ngôn ngữ bạn muốn (ví dụ: Vietnamese).
- Chọn dịch vụ dịch: Bạn có thể dùng Google Translate hoặc Microsoft Translator (miễn phí).

### Bước 5: Bắt đầu dịch
- Nhấn nút **Translate** để bắt đầu.
- Phần mềm sẽ tự động dịch từng dòng phụ đề.
- Quá trình này có thể mất vài phút tùy thuộc vào số lượng dòng.

### Bước 6: Kiểm tra và chỉnh sửa
- Sau khi dịch xong, xem lại từng dòng.
- Nếu có lỗi dịch, nhấp vào dòng đó và sửa trực tiếp trong hộp văn bản.
- Bạn có thể nghe video để đảm bảo phụ đề khớp với lời nói.

### Bước 7: Lưu file mới
- Chọn **File** > **Save as** (hoặc Ctrl+Shift+S).
- Đặt tên file mới (ví dụ: movie_vietnamese.srt) và chọn vị trí lưu.
- Nhấn **Save**.

## Lưu ý
- Dịch tự động không hoàn hảo 100%, hãy kiểm tra lại để đảm bảo ý nghĩa đúng.
- Nếu phụ đề có nhiều dòng đặc biệt (như hiệu ứng), chúng có thể không dịch được chính xác.
- Nếu gặp lỗi, thử tải lại file hoặc chọn dịch vụ khác.

Bây giờ bạn đã có file phụ đề đã dịch! Tiếp tục với các hướng dẫn khác để học thêm tính năng.
