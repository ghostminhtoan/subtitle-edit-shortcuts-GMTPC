# Profile 03: Edit Translated Subtitle
## (Sửa file phụ đề đã được người khác dịch)

> **Workflow:** Mở file đã dịch → Sửa lỗi chính tả, format, timing → Save

## 💡 TIPS SỬ DỤNG

1. **Bước 1:** Mở file đã dịch → `Ctrl+Shift+F` để chạy Fix Common Errors
2. **Kiểm tra chính tả:** `Alt+F7` → sửa từng từ sai, `Ctrl+Shift+D` tìm từ lặp
3. **Sửa formatting:** `Ctrl+R` ngắt dòng lại cho đẹp, `Ctrl+M` gộp dòng nếu cần
4. **Sửa timing:** Dùng `Ctrl+Shift+V` (Visual Sync) nếu timing lệch nhiều
5. **Find & Replace:** `Ctrl+H` thay thế từ, `Alt+Shift+M` thay thế hàng loạt
6. **Kiểm tra cuối:** `Ctrl+Alt+E` xem danh sách lỗi trước khi export
